# Security Policy

## 脆弱性の報告 / Reporting a Vulnerability

このリポジトリで脆弱性を発見した場合は、公開の Issue ではなく GitHub の
[Private vulnerability reporting](https://github.com/Shiyo1101/ichigocrate-react/security/advisories/new)
から報告してください。

If you discover a security vulnerability in this repository, please report it
privately via
[GitHub Private vulnerability reporting](https://github.com/Shiyo1101/ichigocrate-react/security/advisories/new)
instead of opening a public issue.

## サポート対象 / Supported Versions

`main` ブランチおよび最新の npm 公開バージョンのみサポートします。

Only the `main` branch and the latest published npm version are supported.
